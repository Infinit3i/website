executive_summary: >
  LummaStealer, also known as LummaC2, is an information stealer offered
  through a malware-as-a-service model on Russian-speaking forums since at
  least August 2022. It is commonly delivered through phishing, fake CAPTCHA
  lures, malicious links, and staged payload chains. The primary impact is the
  theft and exfiltration of user credentials and other sensitive data to
  attacker-controlled command and control infrastructure.

overview:
  type: Infostealer / Malware-as-a-Service
  delivery: Phishing emails, phishing links, fake CAPTCHA pages, malicious downloads, staged follow-on payloads
  capabilities: Credential theft, data staging, exfiltration, remote access enablement, file upload and execution, anti-analysis behavior
  characteristics: >
    LummaStealer is commonly associated with multi-stage delivery chains and is
    often paired with loaders or memory-only payloads such as Peaklight. It is
    frequently observed in fake CAPTCHA and social engineering campaigns and is
    designed to collect and exfiltrate user and system data.

attack_flow:
  diagram: "Nmap → Email → Phishing Link → RDP → File Upload → Run Python Script (Data Staging) → Exfiltration → Remove Files"
  steps:
    - Adversary performs reconnaissance or host discovery
    - Phishing email or lure is delivered to the victim
    - Victim follows a phishing link or fake CAPTCHA workflow
    - Remote access or follow-on access is established through RDP or related means
    - Malicious files are uploaded to the target environment
    - Python or follow-on script is executed for staging collected data
    - Stolen data is exfiltrated to attacker-controlled infrastructure
    - Files or artifacts are removed to reduce forensic visibility

mitigations:
  - name: Endpoint Detection and Response
    value: Implement advanced EDR with behavior-based detections and sandboxing for internet-downloaded executables
  - name: Multi-Factor Authentication
    value: Enforce MFA across local, domain, default, and cloud accounts to reduce the value of stolen credentials
  - name: Security Awareness Training
    value: Conduct regular training on phishing, fake CAPTCHA lures, and social engineering techniques
  - name: Email Security
    value: Deploy robust email filtering to block phishing messages and malicious attachments
  - name: Execution Control
    value: Apply strict software execution policies to block fake installers and unauthorized payload execution
  - name: Application Whitelisting
    value: Allow only legitimate applications and scripts, including controls around mshta.exe abuse
  - name: Firewall and Egress Policy
    value: Restrict or monitor outbound connections over ports 80 and 443 for suspicious processes such as mshta.exe where operationally feasible
  - name: IOC Blocking
    value: Block known indicators of compromise from trusted intelligence sources
  - name: Conditional Access
    value: Block logins from non-compliant devices or unauthorized geographies and IP ranges
  - name: Credential Hygiene
    value: Eliminate default credentials, reduce password reuse, rotate SSH keys where applicable, and enforce strong credential policy
  - name: Account Auditing
    value: Routinely audit domain and local accounts and permissions for unauthorized privileged access paths
  - name: Data Loss Prevention
    value: Use DLP controls to detect and block sensitive data uploads through browsers and web services
  - name: Web Proxy Enforcement
    value: Enforce external communication policy through proxies to prevent use of unauthorized services
  - name: Network Intrusion Detection and Prevention
    value: Use network signatures and anomaly detection to identify adversary malware traffic and suspicious exfiltration patterns

LummaStealer, also known as LummaC2, is an information stealer offered through a malware-as-a-service model on Russian-speaking forums since at least August 2022. It is commonly delivered through phishing, fake CAPTCHA lures, malicious links, and staged payload chains. The primary impact is the theft and exfiltration of user credentials and other sensitive data to attacker-controlled command and control infrastructure.
